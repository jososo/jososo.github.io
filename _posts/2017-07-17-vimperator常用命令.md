## 命令
```
:q         --退出
:wq        --退出and save tab
:res[tart] --restart
:winonly   --关闭除当前窗口外的所有窗口
:winclose  --关闭窗口 
:bmark     --添加入收藏夹
:qmark     --打开书签
```

## 按键
```
/         非ignore下输入查找关键字
n         /之后esc按n能找到下一个关键字
o         在当前tab打开链接
t         跳转到新tab打开链接
w         在新窗口打开链接（本机不可用）
y         复制当前网址到剪贴板
p         它的含义是put，火狐即可自动转入复制的网址中
D         关闭前一个tab
r|R       refresh
gh        本tab回到主页
shift+esc 停用vimoperator
gf        查看源码，再次gf回来
gH        新tab打开主页
gu        到二级域名根
gU        到一级域名根
j         向下滚动一行
k         向上滚动一行
h         向左滚动
l         向右滚动1列
gg        查看当前打开网页的最上面部分


```
